The proxy will allow you to access files underneath /mnt/data/services/apache-tomcat-8090/webapps/ROOT/addama/ui/ on host informatics.42stories.com via a proxy instead of the Addama Registry's support for static files.

The Addama Registry's support for static files caches files in memcache which is generally a good thing, unless you are actively modifying your static files and want to see your changes.
